App: MyBookLibrary

This application was built by Team ARK.
Credits to developers Alex Nam Kim, Richard Buccat, and Kenichi Suragai.


MyBookLibrary is an application built to revolutionize the book industry. 
It was designed to create a community of book lovers, 
and to make reading a more social and enjoyable experience for the modern reader-base.

User login information is stored on a firebase server.
API: Google Books
